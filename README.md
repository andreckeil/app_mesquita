# app_mesquita
Programa para microcontroladores ESP32 desenvolvido para a matéria de Fenômenos Térmicos e Ondulatórios do professor Alexandre Mesquita

Programa desenvolvido em C/C++ utilizando a biblioteca WiFi.h, que utiliza requisições web para montar uma página em HTML para o cliente. Este código é um código extremamente simples com o princpal objetivo de servir como uma "prova de conceito".

Para rodar este código, é necessário utilizar a biblioteca do esp32 para a Arduino IDE, junto com a biblioteca WiFi. 

No futuro, gostaria de implementar o armazenamento das páginas na memória do ESP utilizando SPIFFS, e utilizar a biblioteca WebServer para conseguir gerar uma rede wifi própria do esp.
